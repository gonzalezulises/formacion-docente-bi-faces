## Descripcion

Este modulo es **opcional y asincronico**. Esta disenado para docentes que no han utilizado herramientas como Git, GitHub o Google Colab antes del programa.

No tiene sesion en vivo ni evaluacion. Es material preparatorio para que llegues al Modulo 1 con el entorno listo y las ideas basicas claras.

**Si ya usas GitHub y Colab, puedes saltar este modulo.**

---

## Objetivo

Al completar este modulo, podras:

1. Entender que es Git y GitHub y por que se usan en este programa
2. Navegar un repositorio en GitHub y encontrar archivos
3. Abrir y ejecutar notebooks en Google Colab desde GitHub
4. Escribir texto basico en Markdown (para tus entregables)
5. Tener tu entorno de trabajo listo para el Modulo 1

## Contenido

### 1. Git y GitHub — Que son y por que importan

Git es un sistema de control de versiones: registra cada cambio en los archivos de un proyecto. GitHub es la plataforma donde se almacenan proyectos Git en la nube.

**En este programa usamos GitHub para:**
- Almacenar todos los notebooks, datasets y materiales
- Que puedas acceder al contenido desde cualquier lugar
- Que puedas abrir notebooks directamente en Google Colab

**NO necesitas:**
- Instalar Git en tu computadora
- Saber usar la terminal/linea de comandos
- Crear commits, branches ni pull requests

**Solo necesitas:**
- Navegar el repositorio en el navegador
- Hacer clic en los badges "Open in Colab"

### 2. Google Colab — Tu laboratorio en la nube

Google Colab es un entorno gratuito de Google que permite ejecutar codigo Python directamente en el navegador, sin instalar nada.

**Ventajas:**
- Funciona en cualquier computadora con internet
- No requiere instalacion
- Guarda tu trabajo en Google Drive
- Ya tiene instaladas las librerias que necesitamos (pandas, numpy, matplotlib, etc.)

**Como usarlo:**
1. Ve a [colab.research.google.com](https://colab.research.google.com/)
2. Inicia sesion con tu cuenta de Google
3. Abre un notebook desde GitHub (o haz clic en el badge "Open in Colab" en cualquier notebook del curso)
4. Ejecuta celdas con `Shift + Enter`
5. Guarda tu copia en Drive con `Archivo > Guardar una copia en Drive`

### 3. Markdown — Escribir texto en notebooks

Markdown es un lenguaje simple para dar formato al texto. Lo usaras en tus entregables y reflexiones.

| Sintaxis | Resultado |
|----------|-----------|
| `**negrita**` | **negrita** |
| `*cursiva*` | *cursiva* |
| `# Titulo` | Titulo grande |
| `## Subtitulo` | Subtitulo |
| `- elemento` | Lista con vineta |
| `1. elemento` | Lista numerada |
| `[texto](url)` | Enlace |

## Guia paso a paso: Tu primera sesion

### Paso 1: Accede al repositorio
1. Abre tu navegador
2. Ve a: [github.com/gonzalezulises/formacion-docente-bi-faces](https://github.com/gonzalezulises/formacion-docente-bi-faces)
3. Explora las carpetas: `modulo-01-fundamentos/`, `datasets/`, etc.

### Paso 2: Abre un notebook en Colab
1. Navega a `modulo-01-fundamentos/notebooks/`
2. Haz clic en `01_01_ciclo_valor_datos.ipynb`
3. En la parte superior, veras un badge azul: **"Open in Colab"**
4. Haz clic en el badge
5. Se abrira el notebook en Google Colab

### Paso 3: Ejecuta tu primera celda
1. En Colab, haz clic en la primera celda de codigo
2. Presiona `Shift + Enter` (o el boton de "play" a la izquierda)
3. Espera a que aparezca el resultado debajo de la celda
4. Repite con las siguientes celdas

### Paso 4: Guarda tu trabajo
1. Ve a `Archivo > Guardar una copia en Drive`
2. Se creara una copia en tu Google Drive personal
3. Tus cambios se guardan ahi — el original en GitHub no se modifica
