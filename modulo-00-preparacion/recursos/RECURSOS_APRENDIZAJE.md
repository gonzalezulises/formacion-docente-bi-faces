## Ruta de aprendizaje sugerida

Si nunca has usado estas herramientas, sigue este orden (~2 horas total):

1. **Git/GitHub** (20 min): Ve "Hola Mundo — Git y GitHub en 15 minutos" + explora el repositorio del curso en GitHub
2. **Google Colab** (20 min): Ve "Ringa Tech — Google Colab tutorial completo" + abre el Welcome Notebook
3. **Practica** (30 min): Abre un notebook del curso en Colab, ejecuta celdas, guarda en Drive
4. **Markdown** (10 min): Ve "Fireship — Markdown in 100 Seconds" + prueba en una celda de texto en Colab
5. **Checklist** (10 min): Completa la checklist de preparacion del README del modulo

Si ya conoces alguna herramienta, salta esa seccion.
