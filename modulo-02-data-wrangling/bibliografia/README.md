# Bibliografía - Módulo 02: Data Wrangling accesible (Python + SQL)

Este módulo es técnico y práctico. Se centra en las habilidades fundamentales para manipular, limpiar y transformar datos usando Python (Pandas) y SQL. Los participantes trabajan en Google Colab con datasets reales, aprendiendo a cargar, explorar, filtrar, agrupar, combinar y exportar datos. La bibliografía apoya la transición de "pensar con datos" (Módulo 1) a "trabajar con datos" de forma reproducible.

---

## Libros de referencia

### Python para datos

- **McKinney, W. (2022). *Python for Data Analysis* (3ra ed.). O'Reilly Media.**
  - Referencia definitiva del creador de Pandas
  - Cubre: NumPy, Pandas, limpieza, transformación, agregación
  - [Acceso al repositorio con notebooks](https://github.com/wesm/pydata-book)

- **Sweigart, A. (2019). *Automate the Boring Stuff with Python* (2da ed.). No Starch Press.**
  - Enfoque práctico para automatizar tareas cotidianas
  - Capítulos relevantes: archivos CSV, web scraping, Excel
  - [Acceso libre en línea](https://automatetheboringstuff.com/)

- **Downey, A. B. (2015). *Think Python: How to Think Like a Computer Scientist* (2da ed.). O'Reilly Media.**
  - Introducción pedagógica a Python, ideal para principiantes
  - [Acceso libre en línea](https://greenteapress.com/wp/think-python-2e/)

### Calidad y limpieza de datos

- **McGregor, S. E. (2021). *Practical Python Data Wrangling and Data Quality*. O'Reilly Media.**
  - Enfocado específicamente en wrangling y calidad de datos
  - Casos reales de datos sucios y cómo resolverlos
