# Bibliografía - Módulo 02: Análisis Exploratorio, Visualización e Inteligencia de Negocios

Esta sección recopila referencias para análisis exploratorio de datos (EDA), combinación de datos, visualización, SQL, Power BI y storytelling con datos.

## Libros de Referencia Fundamental

### Storytelling y Visualización
- **Storytelling with Data: A Data Visualization Guide for Business Professionals** (2015)
  - Autora: Cole Nussbaumer Knaflic
  - Conecta análisis de datos con narrativa efectiva
  - Casos prácticos para comunicar hallazgos a stakeholders
  - Imprescindible para educadores que entrenan analistas

- **The Visual Display of Quantitative Information** (2da ed., 2001)
  - Autor: Edward R. Tufte
  - Clásico sobre principios de visualización
  - Análisis crítico de gráficos efectivos e ineficaces
  - Filosofía del "data-ink ratio"

- **The Art of Statistics: Learning from Data** (2019)
  - Autor: David Spiegelhalter
  - Estadística narrativa con énfasis en visualización
  - Casos reales de instituciones y gobiernos
  - Accesible para audiencias no técnicas

## Análisis Exploratorio de Datos (EDA)

### Documentación y Guías
- **Pandas Documentation - User Guide**
  - [https://pandas.pydata.org/docs/user_guide/index.html](https://pandas.pydata.org/docs/user_guide/index.html)
  - Funciones para EDA: `describe()`, `value_counts()`, `groupby()`
  - Manejo de valores faltantes

- **Exploratory Data Analysis - Tradición Tukey**
  - John W. Tukey: "Exploratory Data Analysis" (1977)
  - Enfoque visual e iterativo previo a modelado
  - Influencia fundamental en metodología EDA moderna

### Recursos en Línea
- **Kaggle - Exploratory Data Analysis**
  - [https://www.kaggle.com/learn/exploratory-data-analysis](https://www.kaggle.com/learn/exploratory-data-analysis)
  - Micro-curso práctico con datasets reales
  - Duración: 4 horas

- **Real Python - Pandas EDA Tutorials**
  - [https://realpython.com/](https://realpython.com/)
  - Búsqueda: "exploratory data analysis pandas"
