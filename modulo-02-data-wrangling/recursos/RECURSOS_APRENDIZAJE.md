## Ruta sugerida de estudio

Para quienes necesiten prepararse antes del modulo o quieran reforzar durante:

### Python (si nunca has programado)
1. **Dia 1-3**: Kaggle Learn — Python (micro-curso completo)
2. **Dia 4-5**: Ver los primeros 5 videos de Corey Schafer sobre Python
3. **Continuo**: Consultar "Automate the Boring Stuff" para conceptos especificos

### Pandas (una vez domines lo basico de Python)
1. **Dia 1-2**: Kaggle Learn — Pandas (micro-curso completo)
2. **Dia 3-4**: Ver la playlist de Corey Schafer sobre Pandas
3. **Dia 5**: Ver el tutorial completo de Keith Galli como revision
4. **Continuo**: Consultar la documentacion oficial de Pandas como referencia

### SQL (puede hacerse en paralelo con Python)
1. **Dia 1**: Completar SQLBolt (tutorial interactivo)
2. **Dia 2-3**: Ver las primeras 2h del curso de freeCodeCamp
3. **Dia 4+**: Practicar en HackerRank SQL (nivel Easy y Medium)
4. **Continuo**: Consultar W3Schools como referencia rapida
