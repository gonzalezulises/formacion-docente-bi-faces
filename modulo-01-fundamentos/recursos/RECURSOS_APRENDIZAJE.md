# Recursos de Aprendizaje — Modulo 01: Pensar con datos

> Recursos complementarios para profundizar en pensamiento estadistico, sesgos cognitivos, alfabetizacion de datos y metodologia en ciencias sociales.
> Este modulo es **conceptual** — no requiere programacion.

---

## Ruta sugerida de estudio

Para quienes quieran profundizar mas alla del contenido del modulo:

1. **Semana 1**: Ver el TED Talk de Hans Rosling + explorar Gapminder Tools
2. **Semana 2**: Ver 3-4 episodios de CrashCourse Statistics + explorar Seeing Theory
3. **Semana 3**: Ver el video de 3Blue1Brown sobre Bayes + leer un capitulo de Factfulness
4. **Semana 4**: Ver 2-3 clases de Calling Bullshit + revisar Spurious Correlations
5. **Continuo**: Leer uno de los libros recomendados a ritmo propio
