# Bibliografía - Módulo 01: Fundamentos de Python y Análisis de Datos

Esta sección recopila referencias esenciales para el aprendizaje de Python, Jupyter, Pandas introductorio, ética de datos y pensamiento analítico en contextos educativos.

## Libros de Referencia

### Python General
- **Think Python: How to Think Like a Computer Scientist** (2015)
  - Autor: Allen B. Downey
  - Enfoque introductorio y pedagógico ideal para educadores
  - [Acceso libre en línea](https://greenteapress.com/wp/think-python-2e/)

- **Automate the Boring Stuff with Python** (2da ed., 2019)
  - Autor: Al Sweigart
  - Práctico y orientado a automatización de tareas cotidianas
  - [Disponible gratis en línea](https://automatetheboringstuff.com/)

- **Python for Data Analysis** (3ra ed., 2022)
  - Autor: Wes McKinney (creador de Pandas)
  - Referencia fundamental para análisis de datos con Python
  - Cubre: NumPy, Pandas, análisis exploratorio

## Ética de Datos y Responsabilidad

### Referencias Académicas
- **FAIR Data Principles**
  - Wilkinson, M. D., et al. (2016). "The FAIR Guiding Principles for scientific data management and stewardship"
  - *Scientific Data* 3, 160018
  - [https://www.nature.com/articles/sdata201618](https://www.nature.com/articles/sdata201618)
  - Accesibilidad, Interoperabilidad, Reutilización

- **Data Feminism** (2020)
  - Autoras: Catherine D'Ignazio y Lauren F. Klein
  - Examina sesgos en datos y algoritmos desde perspectiva feminista
  - Aplicable a contextos educativos e institucionales

- **UNESCO Guidelines on Artificial Intelligence and Education** (2021)
  - [https://www.unesco.org/en/articles/unesco-guidance-artificial-intelligence-education-2021](https://www.unesco.org/en/articles/unesco-guidance-artificial-intelligence-education-2021)
  - Ética, equidad, transparencia en tecnología educativa

### Educación Basada en Datos
- **Data Literacy: A Users Guide** (2016)
  - Autores: Ellen B. Mandinach y Edith S. Gummer
  - Fundamentos de alfabetización en datos para educadores
  - Marco conceptual aplicable a FACES

- **Data Literacy Project**
  - [https://www.datalit.org/](https://www.datalit.org/)
  - Recursos para desarrollar competencias analíticas en educación
  - Lecciones estructuradas, ejercicios, estudios de caso

## Pensamiento Analítico para Educadores

### Conceptos Fundamentales
- **The Art of Statistics: Learning from Data**
  - Autor: David Spiegelhalter
  - Enfoque narrativo sobre estadística y toma de decisiones
  - Casos reales educativos e institucionales

- **Thinking, Fast and Slow**
  - Autor: Daniel Kahneman
  - Sesgos cognitivos y toma de decisiones
  - Relevancia para análisis crítico de datos

### Metodología
- **Exploratory Data Analysis (EDA)** - Tradición Tukey
  - John Tukey: pionero en análisis exploratorio
  - Enfoque visual e iterativo
  - Antecedente conceptual de pandas
