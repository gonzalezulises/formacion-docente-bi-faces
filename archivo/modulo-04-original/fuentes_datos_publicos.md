## Consejos para Usar Datos Publicos

1. **Verificar la fecha de actualizacion.** Muchos portales tienen datos desactualizados.
2. **Documentar la fuente.** Siempre citar de donde provienen los datos (APA 7).
3. **Validar la calidad.** Revisar valores faltantes, inconsistencias, cambios de metodologia.
4. **Respetar las licencias.** Verificar si los datos son de uso libre o tienen restricciones.
5. **Complementar fuentes.** Un solo dataset rara vez cuenta toda la historia.
6. **Preferir CSV/XLS sobre PDF.** Los datos en PDF requieren extraccion manual (tabula-py puede ayudar).
