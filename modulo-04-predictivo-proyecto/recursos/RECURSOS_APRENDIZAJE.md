## Ruta de aprendizaje sugerida

Para aprovechar estos recursos en el orden mas efectivo:

1. **Fundamentos estadisticos**: StatQuest (videos 1-5) + capitulos 2-4 de ISLR
2. **Algebra lineal (opcional pero recomendado)**: 3Blue1Brown playlist
3. **Primera practica**: Kaggle Learn Intro + Intermediate (~7h total)
4. **Regresion en profundidad**: Scikit-learn docs de Linear Models + Machine Learning Mastery
5. **Clasificacion y evaluacion**: StatQuest Confusion Matrix + Scikit-learn Model Evaluation
6. **Proyecto end-to-end**: Videos de Ken Jee y Krish Naik + metodologia CRISP-DM
7. **Proyecto integrador**: Elegir dataset de Kaggle/Data.gov, aplicar CRISP-DM completo
8. **Profundizacion (post-curso)**: Andrew Ng Specialization + libro de Geron
